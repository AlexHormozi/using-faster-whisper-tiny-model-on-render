# Faster Whisper Real-Time Transcription

This project provides real-time transcription of audio using the Faster Whisper Tiny model. It captures live audio and sends transcriptions via WebSocket.

## Requirements

- Python 3.9
- Install dependencies using `pip install -r requirements.txt`

## Running the App

1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`

The server will run at `ws://localhost:8765` for WebSocket connections.
